## 使用方式

- 发送 Command:
    * 加密：`RC/rc一下 [文本]`
    * 解密：`一下RC/rc [RC码]`

- DEMO:
    * RC一下 RC为什么这么NB？
    * `>> ȐĉņþȠƃȒĊňƅŘƇƝƀŘƈȠƄȒĊńbRcƝBřćȠƄȒĊŃƅrȼřčńƄŇþŘȻ  # RC码`
